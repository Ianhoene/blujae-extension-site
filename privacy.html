Privacy Policy

This Privacy Policy explains how Blujae collects, uses, stores, and shares information when users interact with the Blujae Chrome extension.

1. Information users may provide

  • Career profile details such as current role, target roles, experience, and skills
  • Job preferences such as work arrangement, locations, compensation, and dealbreakers
  • Career history, certifications, and education details
  • Resume files uploaded during onboarding
  • Visible job listing information from pages the user chooses to analyze

2. How information is used

  • Generate job fit analyses and related insights
  • Parse resume content and prefill onboarding fields
  • Store user progress and preferences locally in the extension
  • Support extension functionality, troubleshooting, and service reliability

3. Local storage in the browser

Blujae stores certain data in Chrome local storage on the user’s device. This may include onboarding drafts, the saved profile, resume-derived structured data, and extension state needed for the product to work between sessions.

4. Backend processing

When a user requests job analysis or resume parsing, Blujae sends relevant payloads to backend services for processing. Depending on the feature used, those payloads may include profile details, extracted job listing text, resume files, and resume-derived context.

5. Website access

Blujae may access page content on websites the user visits in order to detect job listings and extract job information for analysis. The extension is intended for use on pages where the user chooses to engage the product workflow.

6. Data sharing

Blujae shares information with backend service providers only as needed to deliver job analysis and resume parsing functionality. This policy does not state that Blujae sells personal information.

7. Data retention

Data stored in Chrome local storage remains on the user’s device until the user clears it, removes the extension, or the extension overwrites that data. Backend retention practices should be reviewed against the live production setup before store submission.

8. User choices

  • Choose whether to upload a resume
  • Edit or replace profile information inside the extension
  • Remove the extension to stop future collection from extension use
  • Clear extension data from Chrome storage through browser controls

9. Security

Blujae uses reasonable measures to protect information in transit and within the extension workflow. No method of transmission or storage is guaranteed to be completely secure.

10. Changes to this policy

This policy may be updated from time to time. The latest version will be posted in this repository with a revised effective date.

11. Contact

Privacy or support requests should be directed through the publisher contact method used in the Blujae Chrome Web Store listing or associated public repository.
